# Student Activities Repository

## Description
Student Activities Repository Web is a frontend web application developed using HTML, CSS, and JavaScript.  
The system helps students explore and manage university activities easily.

## Features
- Event Booking
- Event History
- Calendar of Events
- Student Registration
- Contact Us Page

## Technologies Used
- HTML
- CSS
- JavaScript

## Pages
- Home
- Register
- Book Event
- Event History
- Calendar
- Contact Us
